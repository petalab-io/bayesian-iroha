# Hockey Spatial Analysis: 空間統計による真の貢献度可視化プロジェクト

## 1. ビジネス課題と背景
従来のセーブ率（Saved %）は、キーパーが直面する「状況の難易度」を無視していました。本プロジェクトでは、ベイズ空間モデリングを用いて、以下の意思決定を支援します。

- **適正評価（ROI）**: 「難しいシュートを多く防いでいるが、見かけの数字は悪い選手」という真の資産を見極めます。
- **リスク管理**: 運による一時的な好成績か、実力による継続的な貢献かを HDI を用いて切り分けます。

## 2. 戦略的フレームワーク（3層フィルタリング）
分析結果は、以下の基準でビジネス判断（起用・契約）に繋げます。

1. **HDI (統計的実証)**: 選手の GSAx（期待以上のセーブ数）の分布が 0 を超えているか。
2. **ROPE (実質的等価性)**: 1試合あたり ±0.05 点程度の差は「誤差」として無視し、それを超える傑出した才能のみを抽出します。
3. **ROI (経済的評価)**: 防いだ1ゴールが、チームの勝利確率および収益にいくら寄与するかを算出します。

## 3. 技術的アプローチ
- **データソース**: [MoneyPuck.com](https://moneypuck.com/) のシュート座標データ。
- **使用モデル**: 階層ポアソン/二項回帰モデルに空間スプラインを統合。
- **実装思想**: `pymc_v5_best_practices.md` に基づく非中心化実装。
