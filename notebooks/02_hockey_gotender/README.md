# Hockey Spatial Analysis: 空間統計による真の資産価値（GSAx）可視化プロジェクト

## 1. 経営課題と分析の目的
従来のセーブ率（Saved %）のみによる評価では、ゴテンダーが直面する「シュート地点の難易度」が考慮されず、真の貢献度を見誤るリスクがありました。本プロジェクトでは、ベイズ階層モデルを用いて**「運」と「実力」を分離**し、選手獲得・契約におけるROIを最大化することを目的とします。

- **資産の適正評価**: 期待失点減少数（GSAx）を算出し、守備の真の核となる選手を特定します。
- **投資判断の科学化**: 1ゴール阻止の価値を100万円と定義し、年俸交渉の合理的根拠（ROI）を提示します。

## 3. 意思決定フレームワーク（3層フィルタリング）
本プロジェクトでは、分析規約に基づき以下の3段階で評価を行っています。

1. **HDI (統計的実証)**: 94% HDI（確信範囲）がリーグ平均（0点）をどの程度上回っているか。
   - 【定義】確率的に最も尤もらしい値の範囲。【直感】「94%の確率でこの範囲に収まる」という自信のスコープ。
2. **ROPE (実質的等価性)**: 1試合あたり ±0.05 点以内を「実務上の誤差」とし、それを超える選手のみを「傑出」と判定。
   - 【定義】実質的に差がないとみなす領域。【直感】ビジネスインパクトを伴わない微差をノイズとして切り捨てるフィルター。
3. **ROI (経済的評価)**: GSAxを金額換算し、HDI Lower（最悪のシナリオ）でも投資価値があるかを確認。
   - 【定義】投資に対する利益率。【直感】「この選手にいくら払えば、いくら分の失点を防いで得をするか」の勝算。

## 4. 技術的アプローチ
- **データ**: [MoneyPuck.com](https://moneypuck.com/) の2024年シュート座標データ（x, y）。
- **モデル**: 空間スプライン基底関数を用いた階層ベイズ・ベルヌーイ回帰モデル。
- **実装**: PyMC v5 を使用。非中心化（Non-centered Parameterization）により、データが少ない選手でも安定した推論を実現。
